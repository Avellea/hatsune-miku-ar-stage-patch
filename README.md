# MikuAR

PlayStation Vita plugin to enable running `PCSC90032` at any date or time.

Made by Avellea

# Building
```bash
$ mkdir build
$ cd build
$ cmake ../
$ make
```

# Installing
Place the following at the bottom of `ur0:tai/config.txt`
```
*PCSC90032
ur0:tai/mikuar.suprx
```
